## Getting Started

1. Run Docker: ```sudo dockerd```
2. Run Docker container: ```sudo docker compose up -d```
3. Run development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:8000](http://localhost:8000) with your browser to see the result.
